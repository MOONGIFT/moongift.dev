# MOONGIFT.dev

開発者向けのリソースをIssueにまとめているサイトです。

**自由にコンテンツを追加できます！**

## 主なリソース

- オープンソース・ソフトウェア
- ソフトウェア
- Web API
- Webサービス
- ハードウェア

## ブックマークレット

[こちらのブックマークレット](htts://moongift.dev/bookmarklet.html)を使うと、リソースが手軽に追加できます。

## 注意点

- リソースを同時に登録するとIssueが被ります。その後は古いIssueが優先されます。
- OSSは試用していません。また、コメントは各自の意見であり、内容を保証するものではありません。
