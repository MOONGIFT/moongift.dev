[Captions](https://www.captions.ai/)は、AIを活用したクリエイティブスタジオです。動画や字幕の作成、音声の置き換えなどAIを利用したクリエイティブ作業を支援します。

[![CaptionsのWebサイト](/assets/images/captions.jpg)](https://www.captions.ai/)

<!--more-->

## 特徴

Captionsの特徴は以下の通りです。

- アバタージェネレータ
- 動画編集
- 字幕作成
- 音声の置き換え
- ノイズ除去

Captionsでは、自分の声を使って別な言語で話す動画を生成できます。また、カメラを見ていなかった動画を、カメラを見ているように加工することもできます。アバターは主に製品紹介や広告などでの利用を想定しているようです。

## API

CaptionsはAPIを提供しています。提供されているAPIは以下の通りです。

- AI Creator API
- AI Translate API
- AI Ads API
- AI Edit API
- AI Twin API
- Lipdub API

[API — Captions](https://www.captions.ai/api)
