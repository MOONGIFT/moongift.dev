## ライブラリ

ライブラリは以下の言語に対応しています。

- JavaScript
- Ruby
- PHP
- Java
- Rust
- Python
- Golang
- .Net
- Swift
- C++

JavaScriptの実装例を以下に示します。

```javascript
import { getJson } from "serpapi";
const response = await getJson({
  engine: "google",
  api_key: API_KEY, // Get your API_KEY from https://serpapi.com/manage-api-key
  q: "coffee",
  location: "Austin, Texas",
});
console.log(response);
```
